# The Project

Boilerplate project project to help strat new project that include :

- docker
- vite
- vue3
- typescript
- pinia
- vitest (and vue test utils)
- saas
- windiCSS
- eslint/prettier

### Pull Docker images

```bash
  yarn images
```

### Run container and launch dev server

```bash
  yarn up
```

### Kill container

```bash
  yarn down
```

### Reboot the app

```bash
  yarn reboot
```

### Run tests

```bash
  yarn test
```

### Access container's shell

```bash
  yarn shell
```
